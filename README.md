# 🏥✨ Hospital Patient Registry System (MongoDB Edition)

A modern **Hospital Patient Registry System** built using **Node.js, Express.js, MongoDB Atlas, MongoDB Compass, HTML, CSS, and Mongoose**. The application enables hospitals to register patients, securely store and retrieve patient records from a cloud database, and visually manage and verify the data using **MongoDB Compass**, all through a clean and user-friendly interface.

---

## 🌟 Features

- 🩺 Register New Patients
- ☁️ Store Patient Records in **MongoDB Atlas**
- 📋 View All Registered Patients
- 📅 Record Date of Admission
- 💊 Store Illness Details
- 🔗 Connect to MongoDB Atlas using **Mongoose**
- 🌍 Cloud Database Storage
- 🎨 Clean & Responsive User Interface
- ⚡ Fast and Efficient Patient Record Management
- 🔍 View and Verify Patient Records using MongoDB Compass

## 🚀 How to Run

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/hospital-patient-registry.git
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Update MongoDB Connection String

Open **app.js** and replace the MongoDB connection string with your own MongoDB Atlas connection string.

---

### 4️⃣ Start the Server

```bash
node app.js
```

---

### 5️⃣ Open in Browser

```
http://localhost:3000
```

## 💾 Database

This project uses **MongoDB Atlas** as the cloud database and **Mongoose** as the Object Data Modeling (ODM) library for Node.js.

During development, **MongoDB Compass** was used to:

- 📂 Connect to the MongoDB Atlas cluster
- 👀 View patient collections
- ✅ Verify stored patient records
- 🔍 Inspect database documents visually
